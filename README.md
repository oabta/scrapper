# Scrapper

Collection of data scrapping spiders

## Requirements

Python 3.10+

## Installation

```shell
$ git clone https://github.com/oabta/scrapper.git
$ cd scrapper
$ pip install -r requirements.txt
```

Download all previous question papers:

```shell
$ scrapy crawl nsc_examinations 
```
